# 云计算尖子班 - 同学分享文章汇总

## 一、尖子班是什么？

尖子班是以微信群为组织形式，以**50**人为一个班级上限的，**分享和学习**的纯技术社群。

## 二、成立尖子班的目的

突破自己、分享技术、分享经验、共同成长。

> 我们每一个人都是懒惰的，借助尖子班我们希望除了可以从自身角度督促自己，还能以集体的氛围激励自己学习和分享，最终达到共同成长的目的。
